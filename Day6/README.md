# Day 6 - Exploratory Data Analysis (EDA) 📊

## 📌 Task
Perform Exploratory Data Analysis (EDA) on an employee dataset using **Pandas** and **Matplotlib**.

## 🎯 Objective
The objective of this project is to analyze employee data, understand its structure, generate statistical insights, and visualize important information using charts.

---

## 📂 Project Structure

```text
Day6/
├── eda_task.py
├── employee_data.csv
└── README.md
```

---

## 🛠️ Technologies Used

- Python 3.x
- Pandas
- Matplotlib

---

## 📦 Installation

Install the required libraries:

```bash
pip install pandas matplotlib
```

---

## 📊 Dataset

The dataset contains employee information, including:

- Employee ID
- Name
- Department
- Age
- Salary
- Experience

---

## 📚 Topics Covered

- Reading CSV files using Pandas
- Displaying dataset information
- Statistical summary (`describe()`)
- Checking missing values
- Checking duplicate records
- Data filtering
- GroupBy analysis
- Correlation analysis
- Data visualization using Matplotlib

---

## 📈 Charts Created

- 📊 Bar Chart – Average Salary by Department
- 📈 Line Chart – Salary Trend

---

## ▶️ How to Run

Run the following command:

```bash
python eda_task.py
```
